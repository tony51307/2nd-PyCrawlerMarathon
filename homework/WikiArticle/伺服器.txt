<p><b>伺服器</b>（英語：<span lang="en">server</span>）指：
</p>
<p>伺服器通常以網絡作為媒介，既可以通過<a class="mw-redirect" href="/wiki/Intranet" title="Intranet">區域網路</a>對內提供服務，也可以通過<a class="mw-redirect" href="/wiki/Internet" title="Internet">廣域網路</a>對外提供服務。伺服器的最大特點就是其強大的運算能力，即使是一部簡單的伺服器系統，通常也至少也要有兩顆處理器以構成<a href="/wiki/%E5%AF%B9%E7%A7%B0%E5%A4%9A%E5%A4%84%E7%90%86" title="對稱多處理">對稱多處理</a>架構，使其能在短時間內完成大量工作，並為大量用戶提供服務。20世紀90年代之後，隨著<a href="/wiki/%E8%B0%83%E5%88%B6%E8%A7%A3%E8%B0%83%E5%99%A8" title="數據機">數據機</a>技術的發展，網際網路由窄頻的電話撥接，升級成為寬頻資料，這代表著以資訊高速公路為象徵的網路新時代來臨。網際網路普及同時改變了電腦使用者習慣，更大大普及網路聯絡傳訊的方式，從文字到圖片，再到影片，伺服器所能完成的工作也越來越複雜；而雲端、大資料時代造就了各種新型態行業，如網路商店、網路電商、網路拍賣、網路銷售、網路遊戲、網路設計及架設，以及越來越普遍性的雲端資料庫或備份庫。標準伺服器（server）及檔案伺服器（NAS）的普及正在時時最佳化及改變現有人類的生活。
</p>
<p>有時，這兩種定義會引起混淆，如<a class="mw-redirect" href="/wiki/%E7%BD%91%E9%A1%B5%E6%9C%8D%E5%8A%A1%E5%99%A8" title="網頁伺服器">網頁伺服器</a>。它可能是指用於<a class="mw-redirect" href="/wiki/%E7%BD%91%E7%AB%99" title="網站">網站</a>的電腦，也可能是指像<a class="mw-redirect" href="/wiki/Apache" title="Apache">Apache</a>這樣的軟體，執行在這樣的電腦上以管理網頁組件和回應<a href="/wiki/%E7%BD%91%E9%A1%B5%E6%B5%8F%E8%A7%88%E5%99%A8" title="網頁瀏覽器">網頁瀏覽器</a>的請求。
伺服器的構成包括處理器、硬碟、記憶體、系統匯流排等，和通用的電腦架構類似，但是由於需要提供高可靠的服務，因此在處理能力、穩定性、可靠性、安全性、可延伸性、可管理性等方面要求較高。
在網路環境下，根據伺服器提供的服務類型不同，分為檔案伺服器，資料庫伺服器，應用程式伺服器，Web伺服器等。
</p>
<p>伺服器作為<a href="/wiki/%E7%A1%AC%E4%BB%B6" title="硬體">硬體</a>來說，通常是指那些具有較高計算能力，能夠提供給多個用戶使用的電腦。伺服器與<a class="mw-disambig" href="/wiki/PC" title="PC">PC</a>機的不同點很多，例如<a class="mw-disambig" href="/wiki/PC" title="PC">PC</a>機在一個時刻通常只為一個用戶服務。伺服器與主機不同，主機是通過<a href="/wiki/%E7%B5%82%E7%AB%AF" title="終端">終端</a>給用戶使用的，伺服器是通過網路給<a href="/wiki/%E5%AE%A2%E6%88%B7%E7%AB%AF" title="客戶端">客戶端</a>用戶使用的，所以除了要擁有終端裝置，還要利用網路才能使用伺服器電腦，但用戶連上線後就能使用伺服器上的特定服務了。
</p>
<p>和普通的<a class="mw-redirect" href="/wiki/%E5%80%8B%E4%BA%BA%E9%9B%BB%E8%85%A6" title="個人電腦">個人電腦</a>相比， 伺服器需要連續的工作在7×24小時環境。這就意味著伺服器需要更多的穩定性技術<a href="/wiki/%E5%8F%AF%E9%9D%A0%E6%80%A7%E3%80%81%E5%8F%AF%E7%94%A8%E6%80%A7%E5%92%8C%E5%8F%AF%E7%B6%AD%E8%AD%B7%E6%80%A7" title="可靠性、可用性和可維護性">RAS</a>，比如支援使用<a href="/wiki/%E7%BA%A0%E9%94%99%E5%86%85%E5%AD%98" title="糾錯記憶體">ECC記憶體</a>。
</p>
<p>根據不同的計算能力，伺服器又分為工作群組級伺服器，部門級伺服器和企業級伺服器。伺服器作業系統是指執行在伺服器硬體上的作業系統。伺服器<a href="/wiki/%E6%93%8D%E4%BD%9C%E7%B3%BB%E7%BB%9F" title="作業系統">作業系統</a>需要管理和充分利用伺服器硬體的計算能力並提供給伺服器硬體上的軟體使用。
</p>
<p>現在，市場上有很多伺服器作業系統。<a class="mw-redirect" href="/wiki/%E7%B1%BBUnix" title="類Unix">類Unix</a>作業系統是<a class="mw-redirect" href="/wiki/Unix" title="Unix">Unix</a>的後代，大多為作伺服器平台
設計。常見的此類<a class="mw-redirect" href="/wiki/%E7%B1%BBUnix" title="類Unix">類Unix</a>作業系統有各種<a href="/wiki/Linux%E5%8F%91%E8%A1%8C%E7%89%88" title="Linux發行版">Linux發行版</a>（如<a href="/wiki/Red_Hat_Enterprise_Linux" title="Red Hat Enterprise Linux">紅帽企業Linux</a>、<a href="/wiki/SUSE" title="SUSE">SUSE</a>）、<a class="mw-redirect" href="/wiki/AIX" title="AIX">AIX</a>、<a href="/wiki/IRIX" title="IRIX">IRIX</a>、<a href="/wiki/FreeBSD" title="FreeBSD">FreeBSD</a>、<a href="/wiki/Solaris" title="Solaris">Solaris</a>、、<a href="/wiki/OpenBSD" title="OpenBSD">OpenBSD</a>、<a href="/wiki/NetBSD" title="NetBSD">NetBSD</a>、和SCO OpenServer。<a href="/wiki/%E5%BE%AE%E8%BD%AF" title="微軟">微軟</a>也推出了<a href="/wiki/Microsoft_Windows" title="Microsoft Windows">Microsoft Windows</a>伺服器版本，如<a href="/wiki/Windows_NT_4.0" title="Windows NT 4.0">Windows NT 4.0 Server</a>、<a class="mw-redirect" href="/wiki/Windows_2000_Server" title="Windows 2000 Server">Windows 2000 Server</a>、<a href="/wiki/Windows_Server_2003" title="Windows Server 2003">Windows Server 2003</a>、<a href="/wiki/Windows_Server_2008" title="Windows Server 2008">Windows Server 2008</a>、<a href="/wiki/Windows_Server_2008_R2" title="Windows Server 2008 R2">Windows Server 2008 R2</a>、<a href="/wiki/Windows_Server_2012" title="Windows Server 2012">Windows Server 2012</a>、<a class="mw-redirect" href="/wiki/Windows_Server_2012_R2" title="Windows Server 2012 R2">Windows Server 2012 R2</a>、<a href="/wiki/Windows_Server_2016" title="Windows Server 2016">Windows Server 2016</a>、<a href="/wiki/Windows_Server_2019" title="Windows Server 2019">Windows Server 2019</a><sup class="reference" id="cite_ref-1"><a href="#cite_note-1">[1]</a></sup>。
</p>
<p>伺服器<a href="/wiki/%E8%BD%AF%E4%BB%B6" title="軟體">軟體</a>的定義如前面所述，伺服器軟體工作在<a class="mw-redirect" href="/wiki/%E5%AE%A2%E6%88%B7%E7%AB%AF-%E6%9C%8D%E5%8A%A1%E5%99%A8" title="客戶端-伺服器">客戶端-伺服器</a>或<a href="/wiki/%E6%B5%8F%E8%A7%88%E5%99%A8-%E6%9C%8D%E5%8A%A1%E5%99%A8" title="瀏覽器-伺服器">瀏覽器-伺服器</a>的方式，有很多形式的伺服器，常用的包括：
</p>
<p>伺服器常見的外型有四種：塔式伺服器、機架伺服器（rack）、<a href="/wiki/%E5%88%80%E9%8B%92%E4%BC%BA%E6%9C%8D%E5%99%A8" title="刀鋒伺服器">刀鋒伺服器</a>（blade server）、機櫃式。
</p>
<p>伺服器常遭到駭客攻擊，所以伺服器都要不定時的去做檢查。
</p>
<p>若無有效防火牆管理，或是未做系統漏洞更新，伺服器本身因為固定IP的關係，更易成為攻擊者的目標，因此新聞常見網路勒索綁架檔案。除了家用主機之外，工作場所的伺服器是不法者所特別注視的目標。
</p>
